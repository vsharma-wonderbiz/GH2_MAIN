# 🚀 Main Pages You Should Create

## 1. 🏠 Dashboard Page (Home)

👉 First page user sees

### Show:

* KPI cards:

  * Efficiency %
  * Hydrogen production
  * Energy consumption
* Active alerts count
* Quick status of assets

👉 Think:

> “Overall health of plant”

---

## 2. 📊 Signal Viewer Page (Your Core Feature)

👉 You already planned this — very important

### Features:

* Select:

  * Asset (Electrolyser 1, 2…)
  * Signal (Voltage, Current, Temp)
* Line graph (time vs value)
* Compare signals (multi-line graph)
* Time filters (1 hr / 24 hr)

👉 This is your **main highlight page**

---

## 3. 📈 KPI Dashboard Page

👉 Dedicated KPI analysis

### Show:

* List of KPIs
* Graphs (trend over time)
* Filters:

  * Date range
  * Asset-wise

👉 More detailed than home dashboard

## 4. 🚨 Alerts / Alarm Page

👉 You already built backend — now UI

### Features:

* Filter by:

# ⚡ Pro Tip (Very Important)

Don’t just create pages — **connect them**

Example:

* Dashboard → click asset → Asset Detail
* Asset → open signals → Signal Viewer
* Alert → click → show signal graph

👉 This shows **product thinking**
